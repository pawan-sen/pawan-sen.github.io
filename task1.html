What's This?

In this we have made a model so that users can enter their atm pin using the gestures from their hand gestures. This can be done by removing the keypad in the atm and replacing it with a gesture box in which user can insert their hand in the box and enter gestures. This will help in owering the atm frauds because many of the frauds are done by placing a false key pad on the top of original keypad which stores the key in them which can be retrived later.

Technology used

  • tensorflow
  • opencv
