What's This?

Automatic License Plate Detector is software which will automatically detect license plate of a vehicle from it's camera. This has multiple uses for security as it can be used to analyse the road safety violators and offenders. Also can be used in automatic parking system for assigning parking spaces and collecting tolls. This is basically upgradation of previous project but uses YOLO for detection of license plate.

Technology used

  • YOLO
  • opencv
  • tensorflow
